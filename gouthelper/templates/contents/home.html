Welcome to GoutHelper

This is a site to help answer common clinical questions about gout.

GoutHelper has clinical tools and information pages. Clinical tools take information about a patient and/or symptoms to provide useful information. Tools are either Treatment Aids, which recommend a treatment for a specific scenario, or Decision Aids, which provide information about gout and/or whether a certain type of treatment is appropriate.

Get Started

Start by creating a GoutPatient (pseudopatient). If you'd like to save GoutPatients to reference later, create an account. If you know which tool you need, just pick that from the Navigation Bar.

GoutPatients are a way to conslidate information and use multiple GoutHelper tools without duplicating information. They also don't require, and in fact don't allow, any Protected Health Information (PHI).
